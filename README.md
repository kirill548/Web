## Используемые технологии

- **HTML5** — семантическая разметка, валидный код
- **CSS3** — кастомные стили, анимации, адаптивный дизайн

## Страницы сайта

Сайт реализован как одностраничное приложение (SPA). Навигация осуществляется без перезагрузки страницы через JavaScript.

**Главная страница (Home)** содержит приветственный блок с фотографией и кратким представлением, блок «Ключевые навыки» с иконками (Python, HTML/CSS, Нейросети, Волонтёрство, Амбассадор с векторной SVG-иконкой Профессионалитет), а также контактную информацию и форму обратной связи.

**Страница «Обо мне» (About Me)** включает фотогалерею с личными фотографиями, подробную информацию о студенте и его подходе к развитию, информацию об общественной деятельности (заместитель регионального координатора проекта «Твой Ход», заместитель руководителя МГЕР в Учалинском районе, работа в Комитете по молодёжной политике), а также блок с достижениями: победитель VII конкурса инженерных проектов «Инженериада», участник Всероссийского слёта студентов ПОО в Ноябрьске, участник Всероссийского форума студентов в Москве.

**Страница «Внеучебная жизнь» (Extracurricular)** содержит информацию об общественной деятельности с логотипами проектов, описание участия в слёте в Парке «Патриот» с фотогалереей, фотогалерею мероприятий (Всероссийский форум студентов в Москве, Всероссийский слёт студентов ПОО в Ноябрьске, победа в Инженериаде), а также перечень волонтёрских проектов.

## Установка и запуск

Для локального запуска проекта клонируйте репозиторий: `git clone https://github.com/ваш-username/portfolio-site.git`, затем перейдите в папку проекта: `cd portfolio-site`. Добавьте свои фотографии в папку `images` с указанными в структуре именами. Откройте файл `index.html` в браузере двойным кликом или используйте локальный сервер: `python -m http.server 8000`, затем перейдите на `http://localhost:8000`.

Для деплоя на GitHub Pages создайте репозиторий на GitHub, загрузите все файлы проекта, перейдите в Settings → Pages, в разделе Branch выберите `main` и папку `/ (root)`, нажмите Save. Сайт станет доступен по ссылке `https://ваш-username.github.io/portfolio-site/`.

## Замена фотографий

Все фотографии хранятся в папке `images`. Чтобы заменить изображения, откройте папку `images` и замените существующие файлы на свои, сохранив те же имена: `my-photo.jpg` — главное фото, `about-photo1.jpg`, `about-photo2.jpg`, `about-photo3.jpg` — фото в разделе «Обо мне», `patriot-photo1.jpg`, `patriot-photo2.jpg`, `patriot-photo3.jpg` — фото со слёта в Патриоте, `event-moscow.jpg`, `event-noyabrsk.jpg`, `event-ingeniada.jpg` — фото мероприятий, `tvoyhod-logo.png` — логотип проекта «Твой Ход». После замены файлов обновите страницу — новые фотографии отобразятся автоматически. Возможность редактирования фото на сайте отсутствует — сайт предназначен только для просмотра.
